# FARM

> Work in progress: this repository is in an early public stage and will be improved iteratively.

## Overview

This repository contains the code used in the **FARM: Few-shot Adaptive Malware Family Classification under Concept Drift** paper.

This repository is currently a work in progress. The codebase is being actively organized and improved, and more complete documentation and reproducible pipelines will be added over time.

## Reference

**FARM: Few-shot Adaptive Malware Family Classification under Concept Drift**  
arXiv: https://arxiv.org/abs/2601.17907

## Dataset

The experiments use data from **BenchMFC**:  
https://github.com/crowdma/benchmfc
